Smart Warehouse Management System

A full ERP module built for inventory, warehouse tracking, product flow, and automated reporting.

Project Overview

This warehouse management system was designed to streamline stock flow, automate tracking, improve warehouse efficiency, and support accurate real-time reporting. The platform includes product management, stock-in/stock-out processes, barcode scanning, and multi-user roles.

Role

Full-Stack Developer

Tech Stack

  • Laravel 11
  • React (Next.js) 14
  • REST API
  • MySQL, Mongodb

Timeline

Sep 2024 — Otc 2025

Key Features

  • Product & Inventory Management
  • Stock In / Stock Out Workflow
  • Barcode/QR Scanning Support
  • Supplier & Purchase Module
  • Role-Based Access (Admin, Manager, Staff)
  • Real-Time Stock Reports & Dashboard
